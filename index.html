你可以 checkout 这个项目到本地， 使用用 nodejs 启动； 最终要的一点是你要自己修改 请求的 url 和 数据结构

这里应用了 express faker.js or nodemon for watch file;

you can find these packages doc on the npm wen site

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

run:

    $ npm install
    $ node app.js
 or
    $ nodemon app.js    // use nodemon watch files change and will be restart node
